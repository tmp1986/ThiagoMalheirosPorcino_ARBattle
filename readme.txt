Install All programs inside Programs Installers folder following the order
1- Panda 3D 1.7.2 SDK
2- DirectX9
3- SQLAlchemy

After that, you can run inside ARBATTLE SOURCE v.10 > ARBATTLE > run 1.7.2.bat



Verify the path if you face any problem editing the bataplha.py path



Example: 

@echo off 

:Run
	C:\Panda3D-1.7.2\python\python.exe batalha.py

pause
cls
goto run
